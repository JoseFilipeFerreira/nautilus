# Nautilus-wallpaper
Generate a wallpaper overlay based on stats from a discord server

## Getting Started
* Get a discord bot Token and place it into a text file named `auth`
* run `sh wallpaper.sh`
   * Generates and changes your wallpaper using `feh`
